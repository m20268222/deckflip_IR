# 발표자료 인터넷에 올리는 방법 (아주 쉬운 버전)

> 개발을 하나도 몰라도 됩니다. 마우스 클릭만 할 수 있으면 돼요.
> 10분이면 끝나요.

## 올려야 하는 파일 (이 폴더 안에 다 있어요)
| 파일 이름 | 뭐 하는 파일이에요? |
|---|---|
| `index.html` | 심사위원이 링크 열면 제일 먼저 보이는 쇼케이스 무대 |
| `style.css` | 무대의 디자인(색, 크기, 움직임) |
| `script.js` | 버튼, 투표, 카드 뒤집기 같은 동작 |
| `three_scene.js` | 첫 화면의 3D 카드들 |
| `deck.html` | 심사장에서 보신 원래 발표 덱 (마지막 버튼으로 연결돼요) |

**5개 전부** 올려야 해요. 하나라도 빠지면 화면이 이상해져요.

## 따라 하기 (GitHub 웹사이트에서)

1. 인터넷 브라우저(크롬)를 켜요.
2. 주소창에 `github.com` 을 치고 로그인해요.
   (계정: 쇼케이스 주소가 `m20268222.github.io` 로 시작하니까, `m20268222` 계정이에요)
3. 화면에서 **`deckflip_IR`** 이라는 저장소(폴더 같은 것)를 눌러요.
4. 파일 목록 위쪽에 있는 **`Add file`** 버튼 → **`Upload files`** 를 눌러요.
5. 컴퓨터에서 이 폴더(`배포팩_쇼케이스`)를 열고,
   위 표의 **파일 5개를 전부 선택**해서 GitHub 화면 가운데로 **끌어다 놓아요**.
6. "같은 이름의 파일이 있다"고 나와도 괜찮아요. 새 파일로 바꿔치기 되는 게 정상이에요.
7. 아래로 내려서 초록색 **`Commit changes`** 버튼을 눌러요.
8. 1~2분 기다렸다가 폰이나 컴퓨터에서 이 주소를 열어 봐요.
   → `https://m20268222.github.io/deckflip_IR/`
9. 쇼케이스 첫 화면(무중력 카드들)이 나오면 성공! 🎉

## 올린 다음에 꼭 해 볼 것 (2분)
- [ ] 폰 카메라로 **PDF 속 QR을 실제로 찍어서** 열리는지 확인해요.
- [ ] 첫 화면에서 마우스로 **화면을 잡고 움직여** 봐요. 카드들이 따라 돌면 정상이에요.
- [ ] Act 3(손끝 스테이지)에서 **투표를 한 번 눌러** 봐요. 퍼센트가 움직이면 정상이에요.
- [ ] 오른쪽 위 **"▶ 손 놓고 보기"** 를 눌러 봐요. 화면이 알아서 넘어가면서 설명해 줘요.

## 자주 묻는 질문
- **Q. 연습해 보고 싶은데 투표 숫자가 쌓이는 게 싫어요.**
  주소 뒤에 `?poll=test` 를 붙여서 열면 돼요. 연습용 투표함이 따로 생겨서 진짜랑 안 섞여요.
- **Q. 투표 숫자를 0으로 되돌리고 싶어요.**
  수파베이스 관리 화면 → SQL Editor에서 아래 한 줄을 실행하면 돼요.
  `delete from public.deck_votes where poll like 'showcase2026%';`
- **Q. 인터넷이 안 되는 곳에서 열면요?**
  투표만 "미리 보기 집계"로 바뀌고, 나머지 무대는 전부 그대로 움직여요.
- **Q. 폰에서 봐도 돼요?**
  네. 폰에서는 3D가 가벼운 버전으로 자동으로 바뀌고, 나머지는 다 똑같이 동작해요.
